Wand for Minecraft Education Edition JS Minecraft

Wand for Minecraft Education Edition is a clone of the WorldEdit plugin for java servers

Commands

To use wand you first need to get the item. Type "wand" in chat to get it. Right click with the wand to select start and end points of a selection. You can then use the following commands to manipulate the selected area.


set [block name] [id]

Fills the area with blocks

replace [block name] [id] [new block name] [new block id]

Replaces [block name] with [new block name]

clone

Clones selected area to your current location

pos1 / pos2

Sets the first / second point for the selection

clear

Replaces area with air

drain

Replaces all water in the selected area with air

flood

Replacess all air in the selection with water

kill-items

Kills all items

kill [entity name]

Kills all [entity name] entites
