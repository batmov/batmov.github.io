Ostrzeżenia

Edycja - Można kopiować jedynie wizytówki

Występuje gdy użytkownik próbował skopiować bazę danych lub folder (kopiować można jedynie wizytówki)


Edycja - Nie można zmienić nazwy bazy danych

Występuje gdy użytkownik próbował zmienić nazwę bazy danych


Edycja - Nie można usunąć bazy danych

Występuje przy próbie usunięcia bazy danych

Edycja - Ścieżka jest zbyt długa

Występuje gdy użytkownik próbował utworzyć obiekt w folderze którego ścięzka jest dłuższa niż 240 znaków.
Rozwiązanie: Przenieś bazę danych do wyższego folderu lub do dysku (np. C:\Database)

Edycja - Ten plik już istnieje

Występuje gdy obiekt o takiej samej nazwie jak obiekt tworzony przez użytkownika już istnieje

Okno - Plik ustawień nie istnieje lub został niepoprawnie zapisany

Występuje przy korupcji pliku ustawień, edycji lub usunięciu go lub aktualizacji do innej wersji programu
Rozwiązanie: Menedżer wizytówek automatycznie zapisze poprawną wersję pliku ustawień

Błędy

Jeśli aplikacja Menedżer wizytówek wywoła błąd, zostanie on zgłoszony za pomocą aplikacji Crash Reporter. Informacje o błędzie należy przesłać pod adres bartoszmorawiec5(at)gmail.com
