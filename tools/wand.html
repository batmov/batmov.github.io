Wand for Minecraft: Education Edition

A plugin for Minecraft: Education Edition that allows you to easily edit your world

To install Wand, open the code editor in-game (press C) and import wand.mkcd

Downloads

Download Wand Version 1.0

Commands and usage

To use wand you first need to get the item. Type "wand" in chat to get it. Right click with the wand to select starting and end point of a selection. You can then use the following commands to manipulate the selected area.

set [block name] [id]

Fills the area with blocks

replace [block name] [id] [new block name] [new block id]

Replaces [block name] with [new block name]

clone

Clones selected area to your current location

pos1 / pos2

Sets the first / second point for the selection

clear

Replaces area with air

drain

Replaces all water in the selected area with air

flood

Replacess all air in the selection with water

kill-items

Kills all items

kill [entity name]

Kills all [entity name] entites
